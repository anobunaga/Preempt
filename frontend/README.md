# Preempt Frontend

A React TypeScript frontend for the Preempt weather anomaly detection application.

## Setup

1. Ensure the Go backend is running on `http://localhost:8080`. Refer to the main Preempt README for backend setup.

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm run dev
   ```

4. Open `http://localhost:5173` in your browser.

## Features

- Health check of the backend
- Fetch current weather data for a given location
- View historical metrics
- Display detected anomalies
- Show alarm suggestions

## API Integration

The frontend proxies API requests to the backend via `/api` path. Ensure the backend handles CORS or run with the proxy.

## Development

- Built with Vite and React
- TypeScript for type safety
- Uses fetch API for HTTP requests

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

```js

        tsconfigRootDir: import.meta.dirname,
      },
      // other options...
    },
  },

```
